## 再利用しやすいコンポーネントの css を考える

### 課題 1

汎用性が求められるボタンコンポーネントなのに、margin を持っていることが問題である。  
このボタンが margin を持つことによって、margin を必要としない場面においてこのコンポーネントを利用することができなくなってしまう。  
汎用性が求められるコンポーネントなのに、ページのレイアウトに依存したスタイルを帯びてしまっている。
